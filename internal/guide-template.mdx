title: "Guide template"
description: "Internal template for intent-driven guides and recipes."
---

Use this template for new intent-driven pages.

## Frontmatter

```yaml
---
title: "Clear task title"
description: "One-sentence summary of the user outcome."
---
```

## Structure

1. Short opening sentence: when to use this guide
2. **Who this is for**
3. **What you will achieve**
4. **Prerequisites**
5. **Steps**
6. **Verify**
7. **Next related tasks**

## Writing rules

- One intent per page
- Keep the page scoped to one outcome
- Link to canonical concept/reference pages instead of duplicating deep explanations
- Use sentence case headings
- Keep examples realistic to products, plans, purchases, and webhooks
