## Prerequisites

Before you begin, ensure you have:

- A SolvaPay account ([create one here](/get-started/create-account))
- Your first product created, with pricing configured during product creation
- An HTTPS MCP endpoint if you plan to enable Hosted MCP Pay

## Hosted MCP Pay flow

### Hosted MCP Pay product (managed MCP monetization)

Choose this when you want SolvaPay to host auth, identity, billing, checkout, and payment for your MCP
server.

- Create product details
- Select plans
- Enable Hosted MCP Pay and configure MCP URL settings

[Create a hosted MCP Pay product](/mcp-pay/create-hosted-mcp-pay-product)

## Shared post-setup checks

After setup:

1. Verify plan visibility and active status on the product details page
2. Test onboarding in sandbox with a new user account
3. Confirm purchases and usage appear in SolvaPay Console

If Hosted MCP Pay is enabled, also confirm the generated MCP server URL works in your MCP client.
