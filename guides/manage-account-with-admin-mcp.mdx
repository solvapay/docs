Use this guide when you want to operate your SolvaPay account directly from an MCP-capable AI assistant.

## Who this is for

- SolvaPay providers with an account
- Teams using Cursor, Claude, or other MCP-capable clients
- Teams that want conversational account operations

## What you will achieve

By the end, you can use the SolvaPay admin MCP Server to manage products, customers, purchases, and checkout sessions from your assistant.

## Prerequisites

- A SolvaPay account
- An MCP-capable AI assistant
- OAuth or API key setup supported by your MCP client

## Steps

1. Start with [Admin MCP Server getting started](/mcp-server/getting-started).
2. Complete [installation](/mcp-server/installation) for your MCP client.
3. Review [available tools](/mcp-server/tools).
4. Use [examples](/mcp-server/examples) to run account-management tasks.
5. Use the MCP tools to generate checkout sessions when needed.

## Verify

- You can authenticate from your MCP client
- Tool calls can fetch and update account data
- You can create a checkout session from the assistant
