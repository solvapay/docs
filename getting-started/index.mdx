## Steps to Launch

Follow these four steps to start accepting payments:

1. **[Create Your Account](/getting-started/create-account)** - Sign up, connect Stripe, and configure your branding
2. **[Choose Your Path](/getting-started/choose-your-path)** - Pick MCP Pay (no-code) or SDK integration
3. **[Test in Sandbox](/getting-started/test-in-sandbox)** - Verify everything works with test data
4. **[Go Live](/getting-started/go-live)** - Complete onboarding and switch to production

## Two Ways to Integrate

SolvaPay offers two approaches to monetize your services:

### MCP Pay (No-Code)

Start by creating a product, then optionally enable Hosted MCP Pay for a no-code MCP setup.
SolvaPay can manage authentication, billing, and access control through a hosted MCP endpoint.

**Best for:** Quick setup, standard OAuth authentication, managed infrastructure

[Get started with MCP Pay](/mcp-pay/overview)

### SDK Integration

Add paywall protection directly in your code using the SolvaPay SDK. Full control over the integration with support for custom authentication flows.

**Best for:** Custom authentication, fine-grained control, integration with existing user systems

[Get started with the SDK](/sdks/typescript/intro)
